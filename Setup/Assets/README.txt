IMAGES POUR LE WIZARD D'INSTALLATION
=====================================

Ce dossier contient les images utilisées dans l'installateur Inno Setup.

Images requises :
-----------------

1. WizardImage.bmp
   - Dimensions : 164 x 314 pixels
   - Profondeur de couleur : 24 bits ou 32 bits
   - Format : BMP non compressé
   - Utilisation : Image latérale gauche du wizard d'installation
   - Recommandation : Utiliser le logo RH Plus ou une image professionnelle

2. WizardSmallImage.bmp
   - Dimensions : 55 x 55 pixels
   - Profondeur de couleur : 24 bits ou 32 bits
   - Format : BMP non compressé
   - Utilisation : Petite icône en haut à droite du wizard
   - Recommandation : Logo RH Plus carré

Note : Ces images sont optionnelles. Si elles ne sont pas fournies,
Inno Setup utilisera les images par défaut.

Comment créer ces images :
--------------------------

1. Ouvrir votre logo dans un éditeur d'image (Photoshop, GIMP, Paint.NET)
2. Redimensionner aux dimensions exactes requises
3. Sauvegarder au format BMP
4. Placer les fichiers dans ce dossier avec les noms exacts ci-dessus

Conseils de design :
-------------------

- Utiliser des couleurs professionnelles cohérentes avec votre marque
- Éviter les images trop chargées (simplicité = professionnalisme)
- Tester sur fond clair et foncé
- S'assurer que le texte reste lisible si présent sur l'image

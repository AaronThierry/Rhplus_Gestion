========================================================================
  GESTION MODERNE RH - NOTES DE MISE À JOUR v1.0.3
========================================================================

Date de publication : 21 janvier 2026

Cette mise à jour corrective résout deux problèmes critiques dans les
modules de gestion des salaires et de saisie par lot.

------------------------------------------------------------------------
CORRECTIONS DE BUGS
------------------------------------------------------------------------

✓ CORRECTION CRITIQUE #1 : Affichage du Net à Payer sur Bulletin Journalier

  Problème résolu :
  - Le bulletin de paie des salaires journaliers affichait 0,00 FCFA
    pour le "Net à Payer" même si le calcul était correct dans l'interface

  Cause :
  - Le modèle de données transmis au générateur PDF ne contenait pas
    la valeur finale du net à payer (SalaireNetaPayerFinal)

  Solution :
  - Ajout de la propriété SalaireNetaPayerFinal au modèle du bulletin
  - Le montant correct s'affiche maintenant sur tous les bulletins PDF

✓ CORRECTION CRITIQUE #2 : Employés CDD dans la Saisie de Paie par Lot

  Problème résolu :
  - Les employés CDD (Contrat à Durée Déterminée) avec une date de fin
    de contrat n'apparaissaient pas dans la liste de saisie par lot
  - Impossible de traiter la paie pour les CDD en fin de contrat

  Cause :
  - La requête SQL filtrait tous les employés avec date_sortie définie
  - Les CDD en période finale étaient exclus à tort

  Solution :
  - Modification du filtre SQL pour inclure les employés dont le contrat
    se termine pendant ou après la période de paie
  - Logique : (date_sortie IS NULL OR date_sortie >= periodeFin)

  Résultat :
  - Tous les employés CDI actifs sont affichés
  - Tous les employés CDD actifs sont affichés
  - Les CDD en période finale de paie sont maintenant inclus

✓ AMÉLIORATION : Arrondi du Net à Payer au 1 FCFA Supérieur

  Précision de l'arrondi :
  - Le net à payer final est arrondi au 1 FCFA supérieur (decimal.Ceiling)
  - Exemples :
    * 45678.50 FCFA → 45679 FCFA
    * 45699.99 FCFA → 45700 FCFA
    * 123456.78 FCFA → 123457 FCFA
